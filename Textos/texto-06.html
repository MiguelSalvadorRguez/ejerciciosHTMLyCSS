Estilos de sangrado

Reglas para hacer el código más legible, mediante su sangrado, espaciado y colocacion de las llaves ({}). los cuatro estilos de sangrado más antiguos del lenguaje de programación C.

Estilo K&r

Es el estilo utilizado en el libro sobre programación C escrito por Kernighan y Ritchie y utilizado también en el kernel de Linux (obra de Ritchie y Thompson)

if (cond) { }

Estilo Allman

Es el estilo utilizado por Eric Allman, creador de muchas utilidades BSD

if (cond)
    {
                
    }
